Project-1: Create a html/css based resposive website

Date: 9/14/2018

Description:


I have always been interested in web developing and I desperately wanted to learn that. So, I started looking all over the internet, and have looked at a lot of projects. With some research I decided to learn through online courses and materials, without taking any paid classes in the school. There are plenty of options available online and are enough to learn html and css basics (atleast I believe the same!).


The first site I looked at is w3school and I believe this is one of the most important one. Yes, I know MDN is also good, but I feel w3schools is more organized. I just skimmed through the website super fast and decided to move on to the YouTube tutorials. I think Quentin's channel is a nice one, if you want to quickly learn the basics. Mmtuts is a great channel for some intermediate stuff. To build this website I mostly followed his tutorials are it turned out great.


After building the prototype of my website, I published it using github-pages, which provides free webhosting for students.


Challenges:


  • Learn HTML and CSS
  • Build a website using html and css only
  • Make this website responsive
